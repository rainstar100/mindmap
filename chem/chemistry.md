# 化学
- 研究分子结构及分子间相互作用规律
# 化学应用
- 生活的各个方面，医药，材料，能源，提高生活质量
# 化学危害
- 消耗自然资源
- 处理不当，污染环境，引起动植物生存环境恶化，使人类感染疾病，受到伤害。
